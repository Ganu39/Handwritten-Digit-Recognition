This project implements a Handwritten Digit Recognition system using a Convolutional Neural Network (CNN). The model is trained on the MNIST dataset, which contains 70,000 grayscale images of digits from 0 to 9.

🚀 Features
Image preprocessing and normalization
CNN-based deep learning architecture
High accuracy digit classification (~98–99%)
Visualization of predictions

🛠 Technologies Used
Python
TensorFlow / Keras
NumPy
Matplotlib

📊 Dataset
MNIST dataset (28×28 pixel grayscale digit images)


📈 Results
The trained model achieves approximately 98–99% accuracy on the test dataset and successfully predicts handwritten digits.
